# FocusFlow - Pomodoro Timer

## Introduction

FocusFlow is a simple Pomodoro Timer web application designed to help users stay focused and productive while working or studying.

The application follows the Pomodoro Technique, a time management method that breaks work into focused intervals followed by short breaks. This helps users maintain concentration, avoid burnout, and improve productivity.

## Features

- Work Timer (25 minutes)
- Short Break Timer
- Long Break Timer
- Start and Pause Functionality
- Reset Timer
- Progress Bar Visualization
- Clean and Responsive User Interface using Bootstrap

## How It Works

1. Select a timer mode:
   - Work
   - Short Break
   - Long Break

2. Click the **Start** button to begin the countdown.

3. The timer will count down based on the selected mode while the progress bar updates accordingly.

4. Click **Pause** to temporarily stop the timer.

5. Click **Reset** to return the timer to its default duration.

6. Switch between timer modes at any time depending on your workflow.
